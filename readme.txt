AI-powered multilingual translation for WordPress with OpenRouter and ChatGPT support. Visual front-end editor with WooCommerce compatibility.

== Description ==

**LinguaPress - AI Translation** is the next-generation WordPress translation plugin powered by AI.

Translate your entire site directly from the front-end using a visual translation interface. With built-in support for **OpenRouter** and **ChatGPT (OpenAI)**, you get high-quality, context-aware translations without limitations.

= Key Features =

* **AI-Powered Translation** - Use OpenRouter or ChatGPT for intelligent, context-aware translations
* **System Prompts** - Control the AI's tone and style (formal, casual, technical, etc.)
* **Smart Glossary** - Define words that should never be translated (brand names, technical terms)
* **Background Translation** - New content is automatically translated when published
* **Visual Editor** - Translate directly from the front-end with live preview
* **SEO Metadata Translation** - Translate meta titles, descriptions, and keywords
* **WooCommerce Compatible** - Full support for WooCommerce products and checkout
* **No Character Limits** - Translate as much as you need with your own API keys

= Supported Translation Engines =

* **OpenRouter** - Access multiple AI models including GPT-4, Claude, Llama, and more
* **ChatGPT (OpenAI)** - Direct integration with OpenAI's API
* **Google Translate** - Classic machine translation option
* **DeepL** - High-quality European language translations

= Pro Features Included =

* Extra languages support
* SEO Pack for URL slug translation
* Automatic user language detection
* Translator accounts
* Navigation based on language
* Browse as other roles

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Settings > LinguaPress to configure your languages
4. Add your API key (OpenRouter or ChatGPT) in Automatic Translation settings
5. Start translating!

== Frequently Asked Questions ==

= Do I need an API key? =

Yes, to use AI translation you need an API key from OpenRouter or OpenAI. You can also use Google Translate with your own API key.

= Is there a character limit? =

No! Unlike other services, you use your own API keys so you only pay for what you use.

= Can I control how the AI translates? =

Yes! Use the System Prompt feature to give the AI specific instructions about tone, style, and terminology.
